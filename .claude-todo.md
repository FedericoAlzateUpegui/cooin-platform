## Common Issues & Solutions

### Metro Bundler Issues
- Port conflicts ‚Üí Kill old instances, restart on 8082
- Cache issues ‚Üí Use `npx expo start --web --clear`
- JSX syntax errors ‚Üí Check all tags properly closed
- Blank screen ‚Üí Check for missing providers in App.tsx

### Backend Issues
- CORS errors ‚Üí Check `.env` has correct port in CORS_ORIGINS
- .env changes ‚Üí Requires full server restart (--reload doesn't watch .env)

### i18n Issues
- Language not changing ‚Üí Check LanguageContext initialization
- Missing translations ‚Üí Check en.json and es.json have all keys
- Component not getting translations ‚Üí Ensure using `t()` function from `useLanguage()`
